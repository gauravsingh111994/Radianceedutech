PHP TRAINING COURSE  (PHP)

Course Duration: 75 Hours

 

Course Availability: ON-CAMPUS, OFF-CAMPUS (WEEKDAYS/WEEKENDS)

 

Pre-requisites:

- Knowledge of HTML and CSS

- Knowledge of Object Oriented Programming Languages (C#, C++, etc.)

- MySQL

 

Course Details: This hands-on PHP Programming Course provides the knowledge necessary to design and develop dynamic, database-driven web pages
using PHP Version 5/Higher. PHP is a language written for the web, quick to learn, easy to deploy and provides substantial functionality required for e-commerce.
This course introduces the PHP framework and syntax, and covers in depth the most important techniques used to build dynamic websites. Students learn how to connect to any ODBC-compliant database, and perform hands-on practice with a MySQL database to create database-driven HTML forms and reports. E-commerce skills including user authentication, data validation, dynamic data updates, and Shopping Cart implementation are covered. Students also learn how to configure PHP and the
Apache Web Server. Comprehensive hands-on exercises are integrated throughout to reinforce learning and develop real competency.
