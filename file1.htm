What are Embedded Systems?

An Embedded System is the one that has computer hardware with software integrated in it as the most significant component.  It is a dedicated computer-based system, may be independent or part of a large system with the following components:

1.     Hardware. It includes processor, timers, I/O devices, ports, memories, etc.

2.     Application Software. To perform multiple tasks at the same time.

3.     Real-Time Operating System (RTOS).  It supervises the application software and provides a mechanism to let the processor run a process as per scheduling and perform the context-switch between different processes.
